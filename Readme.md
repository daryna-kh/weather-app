# Weather App

## Getting Started

Follow these steps to set up and run the project locally.

1. **Clone the repository**

   ```bash
   git clone <your-repo-URL>
   cd <your-project-folder>

   ```

2. **Rename the environment file**

   ```bash
   mv .env-sample .env

   ```

3. **Install dependencies**
   ```bash
   npm install
   ```
